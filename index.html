Features

Built to Defeat Blockchain Analysis

🔑
Stealth Addresses
Each transaction uses a fresh one-time address derived from the recipient's meta-address. Their real wallet is never exposed on-chain.
✂️
Automatic Value Split
Breaks amount fingerprinting. No tool can correlate equal values when the split is random every time.
🔀
Multi-Hop Routing
Each transaction part hops through 2–3 fresh disposable wallets. Each wallet is used once and discarded forever.
🎭
Dummy Transactions
Noise transactions injected between real hops. Blockchain analysis tools cannot distinguish real from fake.
⛽
Gasless Withdraw
Recipients withdraw to their real wallet without needing ETH in the stealth address. The connected wallet pays gas via EIP-191 signature relay.
🔗
Non-Custodial
Smart contract on-chain. No pooling with other users. Your funds, your keys — isolated end-to-end.
